### Tutorial para instalação:

```bash
# Clone este repositório
$ git clone https://github.com/brlouco/Jogo-da-Velha

# Entre na pasta
$ cd Jogo-da-Velha

# Baixe as dependências
$ npm install

# Inicie o servidor
$ node index.js

# Por fim, vá até o navegador e coloque localhost:3000 como url
```
